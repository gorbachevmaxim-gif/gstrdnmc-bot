# gstrdnmc-bot

chat bot Gastrodinamika (GSTRDNMC)

View your app in AI Studio: https://ai.studio/apps/2bc221d4-4140-4126-8fc8-1e9159d3c941

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
