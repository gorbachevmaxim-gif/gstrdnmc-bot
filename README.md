# gstrdnmc-bot

Telegram-бот для велосипедного сообщества «Гастродинамика». Помогает райдерам получать информацию о турах, маршрутах, погоде и находить сухие дороги.

## Возможности

- **AI-ассистент** — отвечает на вопросы о велотурах, маршрутах и еде на основе контекста сообщества
- **Календарь туров** — сезонный календарь всех запланированных велотуров с экспортом в iCal
- **Маршруты на выходные** — подборка GPX-маршрутов с данными о погоде и километраже
- **Обход Komoot** — скачивание GPX-файлов из закрытых платных регионов Komoot
- **Давление в шинах** — ссылка на калькулятор давления для разных типов покрышек
- **Карта ресторанов** — избранные точки старта и финиша в городах туров на Яндекс Картах
- **Поиск сухих дорог** — интеграция с приложением RainFree для поиска погодных «окон»

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/help` | Список всех команд |
| `/manifest` | Манифест сообщества |
| `/rules` | Правила безопасного райда |
| `/calendar` | Календарь туров на сезон |
| `/rides` | Маршруты на ближайшие выходные |
| `/gpx <url>` | Скачать GPX из Komoot |
| `/pressure` | Калькулятор давления в шинах |
| `/resto` | Карта ресторанов и кафе |
| `/komoot` | Коллекции маршрутов |
| `/rainfree` | Поиск сухих дорог |

## Технологии

- **Telegram Bot API** (grammy)
- **Groq API** — AI-обработка сообщений (модель Llama 3.1)
- **Redis** — хранение настроек и кэширование
- **Vite + React** — фронтенд (опционально)

## Запуск локально

**Требования:** Node.js

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Настройте переменные окружения в `.env.local`:
   ```
   GROQ_API_KEY=ваш_ключ_groq
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   REDIS_URL=опционально_url_redis
   ```

   Получить ключи:
   - Groq API: https://console.groq.com
   - Telegram Bot: @BotFather

3. Запустите бота:
   ```bash
   npm run dev
   ```

## Деплой

Проект готов к деплою на Vercel. Настройте переменные окружения в панели Vercel.
